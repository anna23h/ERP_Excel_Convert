## 决策
- **二段式取代单一名单过滤器**：发货范围不再靠员工手填「面单已完成名单」，而由完整天猫导出按履约+面单两道筛出 → 消除「名单过期/漏填」整类故障，且天猫导出本就是必备输入。
- **GUI 选项卡 > 单页滚动**：单页改版试过可滚动容器（`2e85793`），最终弃用，改三标签让每屏更短、日志右侧常驻 —— 矮屏不裁切、日志永远可见。
- **「总获单清单」与「新订单获单清单」并存不合并**：两表语义不同（今日发货全集 vs 仅新订单待获单），员工用途不同，各留一份。
- **双击直跑 .bat 与 exe 并存**：不二选一，按员工机器是否装 Python 分流。
