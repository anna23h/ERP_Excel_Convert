# 同事电脑安装指南（一次性，约 15 分钟）

> 目标：同事以后**只双击一个文件**就能用，且每次自动更新到最新版，无需懂命令行。
> 原理：双击 `Windows双击运行.bat` → 自动 `git` 拉最新代码 → 自动补依赖 → 打开界面。

## 一次性准备（你来做，装一次）

1. **装 Python 3**：https://www.python.org/downloads/
   安装时**务必勾选 `Add Python to PATH`**。
2. **装 Git for Windows**：https://git-scm.com/download/win
   一路默认下一步即可（默认会把 git 加进命令行）。
3. **把项目 clone 到同事电脑**（不要用"下载 zip"，必须 clone 才能自动更新）。
   打开 Git Bash，选个目录（如 `D:\工具`），执行：
   ```bash
   cd /d/工具
   git clone https://github.com/anna23h/ERP_Excel_Convert.git
   ```
4. 进入 clone 出来的文件夹，**双击 `Windows双击运行.bat`** 跑一次。
   首次会自动建运行环境、装依赖（1–2 分钟），完成后弹出界面即成功。
5. 建议：右键 `Windows双击运行.bat` → 发送到 → 桌面快捷方式，同事以后点桌面图标即可。

## 同事日常使用

- **双击 `Windows双击运行.bat`（或桌面快捷方式）** → 自动更新 + 打开界面。
- 产出默认写到 `输出\当天日期\` 下，界面上有"📂 打开输出文件夹"按钮。
- 用完直接关窗口即可。

## 更新怎么生效

- 你在这边改完代码 `git push` 后，**同事下次双击就自动是最新版**，无需任何操作。
- 更新只覆盖程序代码；同事的产出（`输出\`）、原始数据（`raw_data\`）都会原样保留。

## 遇到问题

| 现象 | 处理 |
|---|---|
| 双击后提示"未找到 Python 3" | 重装 Python，安装时勾 `Add Python to PATH` |
| 提示"未检测到 Git，跳过自动更新" | 装 Git for Windows；不影响运行，只是不自动更新 |
| 提示"联网更新失败" | 断网了，会用当前版本继续跑；联网后再双击即更新 |
| 界面报错、异常退出 | 窗口会停住显示错误，**截图发给开发者** |

> 说明：自动更新用 `git fetch` + `git reset --hard origin/main`，只同步代码、不动同事的数据文件。仓库是公开仓库，拉取无需账号密码。
